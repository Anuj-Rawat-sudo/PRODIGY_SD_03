# 📒 Contact Management System

A simple **Contact Management System** built using **Python**, **Tkinter**, and **JSON**. This application allows users to store, manage, and organize contact information through an easy-to-use graphical interface.

## 🚀 Features

* ➕ Add new contacts
* 👀 View all saved contacts
* ✏️ Update existing contacts
* 🗑️ Delete contacts
* 🔍 Search contacts by name
* 💾 Automatically save contacts in a JSON file
* 📂 Load saved contacts when the application starts

## 🛠️ Technologies Used

* Python 3
* Tkinter (GUI)
* JSON
* File Handling

## 📋 Functionalities

* Add contact details (Name, Phone Number, Email)
* Display all saved contacts
* Edit selected contact information
* Delete unwanted contacts
* Search contacts by name
* Store contacts permanently using a JSON file

## 📂 Data Storage

The application stores all contact information in a file named **contacts.json**. The file is created automatically after the first contact is added and is loaded each time the program starts.
